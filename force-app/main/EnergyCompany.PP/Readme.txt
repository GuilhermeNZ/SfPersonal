Nesse arquivo estará descrito todas as Premissas, Regras Funcionais e Regras nao Funcionais, sendo enriquecido ao longo do tempo.

Premissas:
		Tela de login
		Processo de Lead -> Conta
		Tela de compra
			Automaçao para definir automaticamente a melhor filial disponivel para tal cliente ( a mais perta )
		Finalizaçao/entrega para o comprador
		Tela de controle de estaçoes/caminhoes ( manutencao etc... )
		Uma tela para consultar situaçao da entrega ( fazer um dev em AWS que contem um banco de dados... )
		

	Regras Funcionais:
		Cada cliente possui um valor de quota especifico
		Dentro do sistema, possuira diversos tipos de vendedores, cada um podendo apenas acessar determinados cliente/produtos
		Regras para finalizaçao da compra ( validacao de documento etc... )
